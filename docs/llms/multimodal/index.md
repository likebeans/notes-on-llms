## 概述

多模态大模型（Multimodal Large Language Models）是能够处理和理解多种模态数据（如文本、图像、音频、视频等）的模型。

## 主要内容

- **视觉语言模型** - CLIP, LLaVA, GPT-4V 等
- **图像生成** - DALL-E, Stable Diffusion, Midjourney 等
- **音频处理** - Whisper, TTS 等
- **视频理解** - 视频描述、视频问答等
